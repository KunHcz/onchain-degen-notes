## 平台机制

### Bonding Curve
- 类似 PumpFun 的联合曲线
- BSC 链上运行
- 毕业后迁移到 PancakeSwap

### 与 PumpFun 对比
| 特性 | FourMeme | PumpFun |
|------|----------|---------|
| 链 | BSC | Solana |
| Gas 费 | 较低 | 极低 |
| 速度 | 3s | 400ms |
| DEX | PancakeSwap | Raydium |
